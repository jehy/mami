Установка Kali Linux

Поставить данный дистрибутив для проведения пентестов можно большим количеством разных способов,в том числе
  • Установить параллельно основной ОС
  • Установить на загрузочный диск (флешку)
  • Установить в виртуальную машину

Рассмотрим третий вариант как наболее удобный. В качестве среды вирутализации будем использовать удобный и бесплатный VirtualBox, который можно скачать на официальном сайте под любую платформу.

Официальный сайт дистрибутива Kali Linux - https://www.kali.org/downloads/. Однако, там доступны только iso образы для установки. Удобнее будет взять сразу готовые образы от авторов в адаптированом для вашей среды виртуализации виде. Их можно загрузить здесь. Если образ у вас не запустился и выдалась ошибка, связанная с USB - отключите в настройках USB - вам этот порт вряд ли понадобится. Логин по умолчанию - root, пароль - toor.
